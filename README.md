## 📌 Sobre o Projeto
Este espaço é dedicado ao armazenamento dos meus primeiros passos no mundo da programação. Aqui encontram-se exercícios de fixação, desafios de lógica e projetos acadêmicos desenvolvidos em **Java**.

## 🧠 Conteúdos Aprendidos
- **Lógica Fundamental:** Desenvolvimento de algoritmos estruturados.
- **Sintaxe Java:** Declaração de variáveis, tipos primitivos e operadores aritméticos.
- **Controle de Fluxo:** - Estruturas de decisão (`if/else`, `switch`).
    - Estruturas de repetição (`for`, `while`, `do-while`).
- **Manipulação de Dados:** Trabalho com Vetores (Arrays) e Matrizes.
- **I/O:** Interação com o usuário via terminal (`Scanner`).
